+ 浮点型

  + torch.float32
 
    通用训练 / 推理（默认）
 
  + torch.float16
 
    混合精度训练（A100/RTX30+）
 
  + torch.bfloat16

    大模型训练（A100/H100，精度优于 FP16）
 
+ 整型

  + torch.int64
 
    类别索引、长度标记（如标签 [0,1,2]）
 
  + torch.int32

    通用整型计算
 
  + torch.int8/uint8
 
    量化推理（低显存）

+ 布尔型

  + torch.bool

    掩码、条件判断（如 padding_mask）

** 1. 类型转换**

```python3
x = x.to(torch.float16)

x = x.float()
```
