```
x = torch.randn(2, 3, 4)
```

**1. 形状**

```python3
print(x.shape)
```

**2. 维度数**

```python3
print(x.ndim)
```

**3. 数据类型**

```python3
print(x.dtype)
```

**4. 步幅**

```python3
print(x.stride())
```

**5. 存储设备**

```python3
print(x.device)
```

**6. 是否连续内存**

```python3
print(x.is_contiguous())
```

**4. 是否需要梯度**

```python3
print(x.requires_grad)
```
