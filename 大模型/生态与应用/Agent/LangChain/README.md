
**LangChain 是一套专为大语言模型（LLM）应用开发设计的开源框架**，核心目标是解决纯 LLM 能力的局限性 —— 让 LLM 能「连接外部资源」「执行复杂任务流」「具备记忆和工具使用能力」，
最终快速构建出实用的 AI Agent 或 LLM 应用（而非仅停留在简单问答）。

## 1. 核心定位：LLM 应用开发的工具箱

+ 对纯 LLM的「能力扩展层」

  弥补 LLM 无记忆、无工具、无流程的短板

+ 开发者的「脚手架」

  提供现成的组件和模板，无需从零编写「LLM 调用→工具集成→记忆管理→结果生成」的全流程代码；

+ 跨模型 / 跨工具的「统一接口层」

  无论你用 OpenAI、本地部署的模型，还是调用搜索引擎、数据库，都能用一套代码适配

## 2. 核心组件

+ 模型 I/O

  统一封装不同 LLM 的调用接口（OpenAI、本地 Llama、ChatGLM 等），支持提示词模板、输出解析

+ 数据连接

  连接外部数据源（数据库、PDF、知识库、API），实现检索增强生成（RAG）

+ 链（Chains）

  将「LLM 调用 + 工具调用 + 数据处理」串联成线性流程

+ 工具调用（Tools）

  让 LLM 调用外部工具（搜索引擎、代码解释器、邮件发送、支付 API）

+ 记忆

  管理会话上下文（短期记忆）和长期知识（向量数据库存储）

+ Agent 智能体

  让 LLM 自主决策「该做什么→选什么工具→执行→反思结果」
