# 1. RBF 网络的结构与原理

RBF 网络通常包含三层：

（1）输入层

（2）隐藏层

每个神经元对应**径向基函数**，计算输入与**中心点**的距离并映射到高维空间

（3）输出层

## 1.1 径向基函数（RBF）

隐藏层的激活函数是径向基函数，其输出仅依赖于样本 $x$ 与升级元中心点 $c_{i}$ 的距离 $||x - c_{i}||$ (径向对称）。最常用的径向基
函数是**高斯函数**:

$\phi_{i}(x)=exp(-\frac{||x-c_{i}||^{2}}{2\sigma_{i}^{2}})\$

其中：

+ $c_{i}$：第 $i$ 个神经元的中心点

+ $\sigma_{i}$ : 第 $i$ 个神经元的宽度参数

+ $||\dot||$：通常为欧式距离


## 1.2 原理

RBF 网络的前向传播过程分为两步：

1. 隐藏层：计算输入 x 与每个中心点 $c_{i}$ 的距离，通过径向基函数映射为高维特征

$\phi(x)=[\phi_{1}(x), \phi_{2}(x), ..., \phi_{n_{hidden}}(x)]^T$

2. 输出层：对隐藏层输出线性组合，得到最终预测

$\hat{y}=w^{T}\phi(x)+b$

# 2. 训练方法

1. 确定隐藏层参数（中心点 $c_{i}$ 和 宽度 $\sigma_{i}$）

+ 中心点 $c_{i}$: 常用方法是 K-means 聚类，将训练样本聚类为 $n_{hidden}$ 类，没类的质心作为中心点；

+ 宽度 $\sigma_{i}$: 通常根据中心点的分布式自适应算法。例如：

$\sigma_{i}=\frac{d_{max}}{\sqrt{2n_{hidden}}}$

其中 $d_{max}$ 是中心点间的最大距离（确保径向基函数覆盖输入空间）
