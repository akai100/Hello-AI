# 2. 感知机

感知机是由两层神经元组成，输入层接收输入信号后传递输出层，输出层是 M-P 神经元，亦称“阈值逻辑单元”。

感知机能容易地实现**逻辑与**、**或**、**非**运算。因为它们都是线性可分的。但是感知机无法解决**异或**这样简单的非线性可分问题。

要解决非线性可分问题，需考虑使用多层功能神经元。输出层与输入层之间的一层神经元，被称为**隐层**或**隐含层**，隐含层和输出层神经元
都是拥有激活函数的功能神经单元。

**多层前馈神经网络**：每层神经元与下一层神经元全互连，神经元不存在同层连接，也不存在夸张连接。
