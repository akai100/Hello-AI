# 2. 标准化和归一化

标准化通过确保所有行和列在机器学习周静得到平等对待，让数据的处理保持一致。


## 2.1 Z分数标准化

利用统计学里简单的 Z 分数（标准分数）思想。Z 分数标准化的输出会被重新缩放，使均值为 0、标准差 1.


公式： $z=(x-\mu)\theta$

其中：

+ $z$ 是新的值（z分数）；

+ $x$ 是单元格原来的值；

+ $\mu$ 是该列的均值

+ $\theta$ 是列的标准差
